Post-Install Instructions

  • Requires SLDS uploaded as a static resource named SLDS
  • Add the following records to the Soundclip__c custom object:
  • url__c: /resource/PanicButtonAssets/PanicButton/SadTrombone.mp3
    Name: Sad Trombone
  • url__c: /resource/PanicButtonAssets/PanicButton/AtomicBomb.mp3
    Name: Explosion
